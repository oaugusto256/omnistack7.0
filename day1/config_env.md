# OmniStack 7.0 - Development environment

## Steps

1. Install NodeJS via package manager at your current machine.
2. Install yarn package.
3. Install VSCode editor at your machine.
4. Install extensions at your VSCode.
   - Dracula theme
   - Material icons
   - FiraCode font

## Stack

- JavaScript
- NodeJS
- MongoDB
- Express
- React
- React Native
